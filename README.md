## 🌦 기능

- **실시간 날씨 정보:** OpenWeatherMap API를 통해 현재 위치 또는 선택한 도시의 날씨 정보를 실시간으로 표시합니다.
- **다양한 배경 이미지:** 날씨에 따라 다양한 배경 이미지로 앱이 자동으로 변경됩니다.
- **도시 선택 기능:** 특정 도시 목록에서 원하는 도시를 선택하여 날씨를 확인할 수 있습니다.
